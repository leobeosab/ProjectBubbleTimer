# ProjectBubbleTimer

## Track PB time using an application in your MacOS tray!

Simple app created using electron to add time to Project Bubble projects and tasks without using the website.

## Tools Used
* Electron v1.6.6
* Node.JS v7.10.0
* npm v4.2.0
* soon Vue.js

## How to build
1. Clone into an empty directory
2. ```npm install```
3. ```npm run package```
